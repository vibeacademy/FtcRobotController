# Competition Prep Command

You are the **backlog-manager** agent. Prepare for an upcoming competition.

## Arguments

- `$ARGUMENTS` - Competition name and date (optional)

## Instructions

1. **Check milestone status**:
   - What tickets are Done?
   - What's still In Progress?
   - Any blockers?

2. **Verify code readiness**:
   - All PRs merged?
   - Tests passing?
   - No known critical bugs?

3. **Generate checklist**:

### 1 Week Before
- [ ] All "Must Have" features complete
- [ ] Code tested in simulator
- [ ] Autonomous routines validated
- [ ] Driver practice scheduled

### 2 Days Before
- [ ] Code freeze in effect
- [ ] Final build deployed to robot
- [ ] Robot inspection checklist reviewed
- [ ] Pit supplies packed

### Day Before
- [ ] Batteries charged
- [ ] Match schedule reviewed
- [ ] Alliance strategy discussed
- [ ] Travel logistics confirmed

### Competition Day
- [ ] Pre-match routine documented
- [ ] Post-match debrief process
- [ ] Issue tracking for fixes
- [ ] Scouting assignments

4. **Identify gaps**:
   - What's not ready?
   - What's the workaround?
   - What's the risk?

## Output

Provide:
- Checklist with current status
- Risk assessment
- Recommendations for final prep
