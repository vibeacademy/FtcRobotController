# Software Robotics — Lesson Index

A beginner-to-competitive YouTube series on the software side of FTC
robotics, taught from this repo. Each lesson has a video and a matching git
branch frozen at the end-of-lesson code state. Start at Lesson 01; each
branch's `LESSON.md` tells you what to read and links you forward.
